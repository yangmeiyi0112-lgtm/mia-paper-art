# 米亞紙意手作坊 官方網站

米亞紙意手作坊的訂購網站，以金色文青風格設計，支援商品選購與訂單填寫。

## 部署說明

本網站使用 **GitHub Pages** 免費靜態託管。

### 第一次部署步驟

1. 在 GitHub 建立一個新的 Repository（建議命名：`mia-paper-art`）
2. 把這個資料夾裡的所有檔案上傳到該 Repository
3. 進入 Repository → **Settings** → **Pages**
4. Source 選擇 `Deploy from a branch`
5. Branch 選 `main`，資料夾選 `/ (root)`
6. 儲存後等 1-2 分鐘，網址會是：`https://你的GitHub帳號.github.io/mia-paper-art`

### 日後更新內容

- 修改商品：編輯 `index.html` 裡的 `products` 陣列
- 修改文字或樣式：直接編輯 `index.html`
- 更新後重新上傳同名檔案即可，GitHub Pages 會自動更新

## 檔案結構

```
mia-paper-art/
├── index.html      ← 主網站（所有內容都在這裡）
└── README.md       ← 說明文件（本檔案）
```
